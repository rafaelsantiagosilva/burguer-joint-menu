# Burguer Joint Menu 🍔

Como o nome diz, a aplicação consiste em um  backend (API Rest) onde o principal objetivo consiste em ser um site para a realização de pedidos em um restaurante.

## Stack 📝

- Express.js
- Drizzle ORM
- PostgreSQL

## Testes 🧪

O principal foco do desenvolvimento desse projeto foi saber como realizar a implementação de testes automatizados, além de implementar arquitetura limpa.
